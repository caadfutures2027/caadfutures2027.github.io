Topics

CAAD Futures 2027 invites original research papers that examine how artificial intelligence reshapes agency within computational design — the conference theme of Design Agency. While the theme foregrounds questions of agency, we welcome a broad range of contributions across computational design. Contributions are not confined to the theme; we welcome broader research that advances computational design methods, tools, theories, or applications.

Author Guidelines

This page provides essential information for authors. Please review the details below carefully.

Submission Format

Initial Submission (PDF only):

  • Submit a single PDF file prepared using the Springer LNCS template. Download in DOC or LaTeX format.
  • Please read the Instructions for Proceedings Authors (PDF) carefully — in particular, Section 4.2 Length of Papers: full papers are typically 12–15 pages (or more).
  • Submit via Microsoft CMT. You will need to create a CMT account if you do not already have one.

Camera-Ready Submission (ZIP):

  • Upon acceptance, authors will submit a final camera-ready version as a ZIP file containing the source files and PDF, following Springer LNCS formatting guidelines.
  • Retain your original DOC or LaTeX source files. All figures must be submitted as separate image files within the ZIP archive.
  • Ensure consistent British English spelling throughout the manuscript.

How to Submit via Microsoft CMT

Step 1. Create a Microsoft CMT account at cmt3.research.microsoft.com and log in.

Step 2. From the top menu, click All Conferences.

Step 3. In the filter box on the right, type caad.

Step 4. Click Computer-Aided Architectural Design Futures 2027 to enter the conference.

Step 5. In the Author Console, click + Create new submission.

Step 6. Select Paper as the submission type and follow the on-screen instructions.
